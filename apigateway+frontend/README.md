source.txt - файл в який треба прокинути текст з інформацією про викладача його наукові роботи , листи тексти

1_prepare.js - файл для перетворення txt в jsonl , модель сприймає його краще 
(команда для запуску : node 1_prepare.js)

2_train.js - файл який запускає fune-tunning , відправляє навчати модель на сервер опенаі і потім відправляє вам на пошту ID моделі яку можна викоритовувати 
(команда для запуску : node 2_train.js)

3_chat.js - файл для спілкування з вже створенною новою доученою моделю 
(команда для запуску : node 3_chat.js)

2_resume.js - файл якшо шоб створився але грошей під час навчання не вистачило 
(команда для запуску : node 2_resume.js)
